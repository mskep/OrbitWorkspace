# YouTube Downloader Tool

A simple tool to download YouTube videos.

## Features

- Download videos in best quality
- Audio-only option
- Customizable download path
- Format selection

## Setup

This tool requires `yt-dlp` to be installed on your system.

### Installation

1. Install yt-dlp: https://github.com/yt-dlp/yt-dlp
2. Make sure it's in your PATH
3. Restart the application

## Usage

1. Paste a YouTube URL
2. Click Download
3. The video will be saved to your configured download path

## Configuration

You can configure:
- Download path
- Video format
- Audio-only mode

Access settings from the tool's settings page.
